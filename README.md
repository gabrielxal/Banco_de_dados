# Banco de dados
 Este repositório foi criado para centralizar atividades, conteúdos e materiais de estudo relacionados a Banco de dados. Aqui, você encontrará exercícios práticos, scripts para consulta e manipulação de dados, e recursos didáticos que abrangem desde conceitos básicos até técnicas avançadas de banco de dados. 

 - Ferramentas: MySQL, PostgreSQL
